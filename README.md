# Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built with React and Vite.

## Features

- **Play vs AI** - Challenge an AI opponent with smart move logic
- **Two Player Mode** - Play against a friend on the same device
- **Score Tracking** - Keep track of wins, losses, and draws across sessions
- **Responsive Design** - Works on desktop and mobile devices
- **Beautiful UI** - Modern gradient design with smooth animations

## Getting Started

### Prerequisites

- Node.js 16+

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build

```bash
npm run build
```

## How to Play

1. Select your game mode (vs AI or Two Player)
2. Click on any empty square to place your mark
3. Win by getting three marks in a row (horizontally, vertically, or diagonally)
4. Click "Restart" to start a new game
